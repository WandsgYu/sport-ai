# Code of Conduct

## Our standard

Be respectful, constructive, and specific. Discuss ideas and code without personal attacks, harassment, discrimination, intimidation, or disclosure of private information.

## Privacy is part of conduct

Never post personal data, credentials, internal endpoints, production logs, private messages, or non-public organizational information. If sensitive material appears, stop sharing it and notify the maintainer privately.

## Enforcement

The repository owner may edit or remove content, close discussions, reject contributions, or restrict participation when behavior threatens people, privacy, security, or a productive technical discussion.

Report conduct or security concerns privately through the repository's security reporting channel.
